# Mompi API

#### Create database
```
rake db:create
```
#### Drop database
```
rake db:drop
```
#### Apply migrations
```
rake db:migrate
```
#### Rollback last migration
```
rake db:rollback
```
#### Seed the database
```
rake db:seed
```
#### Generate schema file
```
rake db:schema
```
#### Reset the database
```
rake db:reset
```
